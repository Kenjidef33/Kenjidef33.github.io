Ik heb voor deze opdracht telkens een spatie gezet bij elke < waarmee het begint om de structuur grotendeels te behouden

(geordende lijst) of < ul> (ongeordende lijst) element alleen < li> (lijstitem) elementen bevatten als directe kinderen. Hier is wat er verkeerd is:

< ul>

< li>een < /li>

< li>twee < /li>

< ol> (dit is verkeerd) binnen deze code structuur.

Hier zit de fout dus bij deze html fragment.

< jan> < /li>

< piet> < /li>

< li> joris < /li>

< /ol>

< li>vier < /li>

< /ul>

Om dit correct te maken, moet je de geneste lijst binnen een apart < li>-element plaatsen, zoals dit:

< ul>

< li>een< /li>

< li>twee< /li>

< li>

< ol>

< li>jan< /li>

< li>piet< /li>

< li>joris< /li>

< /ol>

< /li>

< li>vier< /li>

< /ul>

Op deze manier is de geneste geordende lijst (< ol>) correct ingebed in een lijstitem (< li>) van de bovenliggende ongeordende lijst (< ul>).
