Het HTML-fragment is niet correct genest, wat betekent dat elke geopende tag moet worden gesloten voordat een andere geopende tag wordt gesloten. In het voorbeeld wordt < strong>-element echter binnen het < span>-element geopend, maar gesloten binnen een ander span>-element, wat leidt tot een onjuiste nestingsstructuur waardoor deze voorbeeld fout is. Om dit te herstellen moet je de html code als volgt schrijven: < span>hier komt < strong>een< /strong> < a href="">link< /a> te staan< /span> Het strong-element is correct genest binnen het eerste span-element en wordt gesloten voordat het tweede < span>-element wordt geopend
