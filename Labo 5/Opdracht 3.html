Hier staat het antwoord van opdracht 3 Labo 5 voor web development 1

Bij de vives website zie je in de Chrome developer tool dat de inline style attribuut te vinden is onder de body tag element. In de foto die onder deze tekst staat is de inline style attribuut blauw aangeduid.

In onderstaande foto zie je een voorbeeld op de vives website waarin een stijlblok aangegeven wordt en hoe die eruit ziet. Het is een style blok omdat het zich tussen een < style > attribuut bevindt die ook nog eens closed genest is.

Bij de vives homepagina zie je ook heel erg veel linken naar andere CSS documenten. Vaak zijn deze aangegeven met stylesheet en een href link naar dat document zoals ook bij de volgende afbeelding ook staat in het blauw als voorbeeld van zo een link naar andere CSS documenten.

Bij miras en de VRT heb je dat ook: bij Miras is de stijlattribuut hier verder gezet bij een < div > element.

Voor een stijlblok is er een hele grote code geschreven zelfs dat aantoont welke stijlen er allemaal gebruikt worden om de pagina overzichtelijker te maken. Maar het wordt ook gezet onder een data type als je kijkt valk boven het in het blauw aangeduide gedeelte.

Tenslotte is er dan ook nog weer verschillende linken die gelegd worden naar andere CSS documenten of stylesheets zoals ze genoemd worden in de developer tool van Chrome.

Bij de VRT wordt zelfs eerst een stijlattribuut gebruikt op iframe name elementje.

De stijlblok in VRT is redelijk klein opgebouwd (zie afbeelding hieronder). Deze is helemaal gewijd aan width en niks anders dus style kan op verschillende manieren dus gebruikt worden.

En als laatste zijn er weer linken te vinden naar andere CSS pagina's waardoor als men daarop klikt men uitkomt op een nieuw artikel.
